# REBA Assessment System

🌐 **Live Demo:** https://reba-assessment-system-ymmz.streamlit.app/
> Note: Online demo runs in 2D mode only.
> For full 3D MiDaS depth estimation, run locally.

## Local Installation (Full Version with 3D MiDaS)

1. Clone this repo
   git clone https://github.com/Yoon1414/REBA-Assessment-System.git
   cd REBA-Assessment-System

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py

## Features
-  2D mode — available online and locally
-  3D MiDaS mode — local only (requires GPU recommended)
-  Batch processing, Single image, Video
-  Export Excel, PDF, CSV
